# 班级意见收集系统 - 本地服务器版

这是一个使用本地 Express.js 服务器和 JSON 文件数据库的班级意见收集系统。

## 功能特点

- ✅ 提交意见（实名或匿名）
- ✅ 管理员查看所有意见
- ✅ 管理员回复和删除意见
- ✅ 实时同步（刷新页面即可看到最新数据）
- ✅ 炫酷的视觉效果和动画

## 安装和运行

### 前提条件

确保已安装 Node.js（建议版本 14 或以上）

### 安装步骤

1. 打开命令行，进入项目目录：
   ```bash
   cd "C:\Users\13268\Desktop\工作\Advices"
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动服务器：
   ```bash
   npm start
   ```

4. 在浏览器中打开：
   - 本地访问：http://localhost:3000
   - 网络访问：http://[你的IP地址]:3000

## 数据存储

所有数据存储在 `database.json` 文件中。首次运行时会自动创建。

## 注意事项

- 服务器必须保持运行才能提交和查看意见
- 如果要给其他人使用，需要在同一局域网内，或者配置端口转发
- 不要关闭命令行窗口，否则服务器会停止

## 如何让其他人访问

### 方法一：局域网访问

1. 查看你的本机 IP 地址：
   ```bash
   ipconfig
   ```
   （找到 IPv4 地址，例如 192.168.1.100）

2. 让其他设备通过 `http://[你的IP]:3000` 访问

### 方法二：端口转发（需要路由器支持）

1. 在路由器设置中配置端口转发规则
2. 将外部端口映射到你的电脑的 3000 端口
3. 其他用户通过 `http://[你的公网IP]:3000` 访问

### 方法三：使用内网穿透工具

推荐使用以下工具之一：
- ngrok（https://ngrok.com）
- cpolar（https://www.cpolar.com）
- frp（需要有自己的服务器）

以 ngrok 为例：
```bash
# 下载并安装 ngrok
ngrok http 3000
```

会生成一个临时的公网地址，分享给其他人使用。

## 故障排除

### 端口已被占用

如果 3000 端口被占用，可以修改 `server.js` 中的 `PORT` 常量。

### 提交失败

检查服务器是否正常运行，浏览器控制台是否有错误信息。

### 无法访问

检查防火墙设置，确保允许 3000 端口通过。
